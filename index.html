⚙️ One-time Setup

An IT admin needs to complete this once. After that, employees just sign in.

  1. Go to portal.azure.com → Microsoft Entra ID → App registrations → New registration
  2. Name it WHS Time Tracker, set account type to Single tenant
  3. Add a redirect URI: Single-page application (SPA) → paste the URL where this file is hosted
  4. Under API permissions, add Microsoft Graph → Delegated: User.Read, Sites.ReadWrite.All
  5. Grant admin consent for your tenant
  6. Copy the Application (client) ID and Directory (tenant) ID into the fields below
  7. In SharePoint, create three lists named exactly: TimesheetEntries, Timesheets, and Expenses

Settings are saved in this browser. Each device needs this done once.
