One-time Setup

An IT admin needs to complete this once.

  1. portal.azure.com → Microsoft Entra ID → App registrations → New registration
  2. Name it WHS Time Tracker, Single tenant
  3. Add redirect URI: Single-page application (SPA) → this page URL
  4. API permissions → Microsoft Graph → Delegated: User.Read, Sites.ReadWrite.All → grant admin consent
  5. Copy Application (client) ID and Directory (tenant) ID below
  6. In SharePoint create lists: TimesheetEntries, Timesheets, Expenses
